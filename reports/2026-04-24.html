🔐 Security & Safety

3 items
Anthropic's Mythos breach was humiliating The Verge

Claude Mythos — which Anthropic described as capable of autonomous zero-day discovery and multi-step exploit chaining, and too dangerous to release — was accessed by unauthorized users through a third-party vendor environment. The group reportedly guessed the model's API URL from Anthropic's URL conventions, aided by an insider at a contractor firm with shared API keys. Anthropic says its own systems are unaffected, but the breach is a significant embarrassment given weeks of public messaging about Mythos's unprecedented danger.

CVE-2026-33626: How attackers exploited LMDeploy LLM Inference Engines in 12 hours Sysdig

A critical SSRF vulnerability in LMDeploy — an AI model-serving toolkit with ~8,000 GitHub stars — was weaponized just 12.5 hours after public disclosure, with no proof-of-concept code available. Attackers used the vision-language image loader to probe AWS metadata services, Redis, MySQL, and other internal targets in an eight-minute session. The rapid exploitation illustrates how detailed CVE advisories now function as de facto exploit blueprints in an AI-assisted threat environment.
